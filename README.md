# ExoHabitAI

## Project Description
ExoHabitAI is an AI-based system that predicts the habitability potential of exoplanets using planetary and stellar parameters. Built with a Hybrid Stacking Ensemble combining XGBoost and Random Forest, the system achieves 99% accuracy with optimized precision-recall balance.
